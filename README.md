### 📊 Gym Members — Statistical Analysis (R)
> Complete statistical study with univariate, bivariate and multivariate analysis

- Linear regression Durée → Calories : **R² = 0.82**
- Pearson correlation Session/Calories : **r = 0.91**
- PCA: 2 components explain **52.3%** of total variance
- Chi-square test: independence between gender and workout type (p = 0.705)
- Libraries: `ggplot2` `FactoMineR` `corrplot` `plotrix`
